## 📌 Project Overview

Taxi Burger Restaurant is a modern **Digital Menu Web Application** that dynamically loads food categories and menu items from an API. It provides an interactive and user-friendly interface for customers to explore food items and manage their cart.

This project focuses on **API integration, dynamic UI rendering, and responsive design** using modern frontend technologies.

---

## ✨ Features

* 📡 **API Integration**

  * Fetches food categories dynamically from an external API
* 🍽️ **Dynamic Menu Display**

  * Categories and food items rendered in real-time
* 🛒 **Cart System**

  * Add items to cart with price calculation
* 🎨 **Responsive UI**

  * Clean and modern design using Tailwind CSS & DaisyUI
* ⚡ **Fast Performance**

  * Built with lightweight and optimized JavaScript

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Tailwind CSS**

* **JavaScript (ES6)**

## ⚙️ How It Works

1. The app fetches category data from:

   ```
   https://taxi-kitchen-api.vercel.app/api/v1/categories
   ```
2. Data is processed using JavaScript
3. UI updates dynamically based on API response
4. Users can interact and add items to the cart
